# chatRoom
基于TCP/IP网络编程的文字聊天系统

## Environment && Platform

| Enviro & Plat | Intro |
| :-: | :-: |
| System | Windows 10 |
| Language | JAVA |
| IDE | Eclipse |

## Execute

* 开启服务端(server.java)，处于等待连接请求状态
* 开启多个客户端(client.java)
* 实现多个客户端之间的文字消息互相发送
* 可以测试客户端 **退出聊天室**、**消除消息** 以及 服务端 **停止服务器**、**退出服务器**功能
